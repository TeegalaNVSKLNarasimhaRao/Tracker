1. Getting started

The app is a single HTML file. There is no installation, no account, no server. Everything runs inside your browser.

  1. Download schedule_tracker_v6.html to your device.
  2. Open it in Chrome (recommended) or any modern browser by double-clicking it.
  3. Bookmark it so you can find it quickly each day.
  4. Always open the same file in the same browser — your data lives there.
Your data is tied to the specific browser on the specific device. If you open the file in a different browser or a different device, it starts fresh. Use Export/Import to move data between devices.
Export your backup at least once a week. It only takes two seconds and protects months of data.

2. Top bar

☾ Dark mode

Switches between light and dark colour themes. Your preference is saved and remembered every time you open the app.

📷 Snapshot

Generates a text summary of today: your focus, water intake, habit completion count, and full schedule. It copies to your clipboard automatically so you can paste it into WhatsApp, a notes app, or email. The snapshot also appears as a preview card below the schedule.

↓ Export

Downloads all your data as a JSON file named habit_backup_YYYY-MM-DD.json. This file contains everything: schedules, habits, tracker data, diary entries, holidays, water logs, focus entries, and pauses.

↑ Import

Loads a previously exported JSON file back into the app. The page reloads with all your data restored. Use this to move data to a new device or to restore after accidentally clearing your browser.

3. Always-visible widgets

Today’s banner

Always shown at the top. Displays today’s full date, which schedule to follow today (auto-detected), a badge showing whether it is a weekday / weekend / holiday, your focus for the day (once saved), and a button to jump directly to today’s schedule.

Today’s focus

A single line where you write your most important priority for today. Press Save. It appears in the banner and in the snapshot. Each day gets its own focus entry.

Water tracker Fixed in v6

Ten droplet icons representing glasses of water (250ml each = 2500ml total for the goal).

  • How to log: Tap any droplet to set your intake to that many glasses. Tap the 4th droplet = 1000ml logged. Tap a lower one to reduce.
  • Auto-tick: When you reach 10 glasses (2500ml), the app automatically marks your 2.5 L water habit as done in the tracker. No manual ticking needed.
  • Goal badge: A green ✓ Goal! badge appears when you hit 2500ml.
  • 7-day bar chart: Below the glasses, a small bar chart shows your water intake for each of the last 7 days. Blue bars = goal hit. Lighter bars = partial.
  • Water streak: Shows how many consecutive days in a row you hit the 2500ml goal, counted backwards from today.
  • Reset: Sets today’s count back to zero if you tapped by mistake.

5. Habit Tracker tab

Track which habits you complete each day of the week.

The table

Each row is one habit. Each column is one day of the week (Monday to Sunday). Tap any circle to cycle through states:

  1. Empty — not logged yet
  2. Green ✓ — done
  3. Grey — — skipped intentionally

Tap again from skip to go back to empty.

Notes on habits

Below each circle is a small + (or 📋 if a note exists). Tap it to add a note for that habit on that day. Quick chips help with common reasons: Done!, Skipped - busy, Skipped - forgot, Partial done, Extra effort, Will catch up.

Month percentage

Each habit shows its completion rate for the last 30 days. This is more honest than just looking at one week.

Paused habits

If you are travelling or going through a period where a habit is genuinely not possible, you can pause it for a date range. Paused days show as P in amber and are completely excluded from your score, streak, and progress. Manage pauses in the Manage Habits tab.

Week navigation

Use Prev / Next buttons to browse past and future weeks. Reset week clears all habit data for the current week (asks for confirmation).

Stats cards

  • This week % — completion rate for the current week
  • Done this week — raw count of habit completions
  • Perfect streak — consecutive days where every non-paused habit was done
  • Total habits — number of habits being tracked

Milestone messages

Special celebration messages appear at 7, 14, and 30 consecutive perfect days.

Weekly summary

Auto-generated analysis at the bottom showing: week score, best day, day that needs work, skipped count, streak, and a message based on your performance.

6. Manage Habits tab New in v6

A dedicated tab for full habit management. This is separate from the tracker.

Habit list

Every habit is shown as a card with its name, description, monthly completion %, and action buttons:

  • ↑ ↓ — Reorder the habit up or down in the list
  • Edit — Change the habit name and description
  • Delete — Permanently removes the habit (tracker data remains but is unlinked)

Add new habit

Tap + Add new habit at the top right. Enter a name and a goal description.

Pause a habit

Tap Pause a habit. Select the habit, enter a from date and to date. The habit is paused for that entire range.

Active pauses

All current pauses are listed at the bottom of this tab. Each shows the habit name and date range with a Remove button. You can now delete pauses directly without editing any files.

9. All Notes tab

A searchable, filterable view of every diary entry you have ever written.

Text search

Type any word or phrase. The list filters instantly. Matching text is shown with context around it.

Jump to specific date

Enter a date in the date picker and tap Go to date. The page scrolls to that entry and highlights it briefly in amber. If no entry exists for that date, a message tells you.

Mood filter

Filter entries by mood. You can combine mood filter with text search. Tap All to remove the filter.

Entry count

Shows how many entries match your current search/filter out of total entries.

Actions

  • Open in diary — switches to the Diary tab and opens that date
  • Edit — opens a modal to edit the text and mood
  • Delete — permanently deletes that entry

10. Holidays tab

Tell the app about days that need a different schedule than the default weekday/weekend logic.

Add single day

Enter a date, a name, and whether it is a Holiday (uses Weekend schedule) or a Working day (uses the appropriate weekday schedule). Press Add.

Add date range New in v6

For a stretch of days like a Diwali break or a project crunch week, enter from and to dates and a name. Press Add range.

Smart auto-skip in ranges:

  • If you are adding a Holiday range, Saturdays and Sundays within the range are automatically skipped (they are already on weekend schedule)
  • Any date already marked as a holiday is also skipped to avoid duplicates
  • Only actual qualifying days in the range get added

Quick add Indian holidays

All major Indian national holidays for 2026-2027 are available as one-tap preset buttons. Holidays you have already added disappear from the list.

How auto-detection works

The app checks the day of the week and any special entries you have added. The today banner and schedule tab automatically reflect the correct schedule for any date.

11. Data, backup and sync

Where your data is stored

Everything is stored in your browser’s localStorage under the key nara_v6. Data stays even if you close the browser or restart your computer. Data is lost if you clear your browser’s site data or use private/incognito mode.

How to back up

Tap ↓ Export in the top bar. A file named habit_backup_YYYY-MM-DD.json downloads. Store this in Google Drive, WhatsApp Saved Messages, or any cloud storage. Do this at least once a week.

How to restore

Open the app on any device. Tap ↑ Import. Select your backup file. The app reloads with all data restored.

Moving to a new device

  1. On the old device: tap Export, save the JSON file
  2. Copy the file to the new device (WhatsApp, email, Google Drive)
  3. Open schedule_tracker_v6.html on the new device in Chrome
  4. Tap Import, select the backup file

Using on any device — GitHub Pages option

If you host the file on GitHub Pages (free), you get a permanent URL like yourname.github.io/tracker. You can open it from any device, any browser, anywhere. See the separate GitHub setup guide for step-by-step instructions.

Even with GitHub Pages, data still lives in each device’s browser separately. Use Export/Import to sync between devices. The URL just means you never need to carry the file around.

13. Quick reference

Keyboard shortcuts (none required, but useful to know)

The app is fully mouse/touch operated. No keyboard shortcuts needed.

Frequently asked questions

Q: My data disappeared.
A: Data is stored per browser. If you opened a different browser or used incognito mode, data is not there. Import your backup file.

Q: Water habit not auto-ticking.
A: Make sure a habit named exactly 2.5 L water exists in the Manage Habits tab. The auto-tick looks for that exact name.

Q: How do I undo a habit tick?
A: Tap the circle again. It cycles: empty → done → skip → empty. Tap twice for skip, three times for empty.

Q: Can I have more than one diary entry per day?
A: No. Each day has one entry. Writing again updates the existing one.

Q: How do I remove a pause?
A: Go to Manage Habits tab. Scroll to Active Pauses at the bottom. Tap Remove next to the pause you want to delete.

Q: The heatmap shows paused days as grey. Is that correct?
A: Yes. Paused days have no score and show as grey. This is correct — they are excluded from your consistency measurement.

Q: Can I use this offline?
A: Yes. Once downloaded, the file works completely offline. No internet needed.

Q: Is there a mobile app?
A: Not a native app. But on Android, open Chrome, open the file, tap the three-dot menu, and tap Add to Home Screen. On iPhone, open Safari, tap Share, then Add to Home Screen. The app then appears on your home screen like a native app.
